# Practice Scores

> Use this template to baseline mock-exam performance across every Security+ domain.

## How to Use This File
- Log each practice run right after you finish while feedback is fresh.
- Capture the set name/source so you can revisit the same pool later.
- Track the specific questions or topics you missed; turn them into study targets.

### Tracking Options
- **Option A (recommended):** create `practice-scores.private.md` in this folder to keep personal data off the shared repo (the `.private.md` suffix is already gitignored).
- **Option B:** fork this repository and commit your updates in your own fork.
